- 用途：展示用户学习进度与最近浏览
- 使用接口：

## 📄 页面：课程总览（courses/index.astro）
- 用途：按分类或关键词浏览课程
- 使用接口：
  - GET /api/v1/courses?category=xxx&search=xxx → 获取课程列表
  - GET /api/v1/categories                      → 获取课程分类列表

## 📄 页面：课程详情页（courses/[slug].astro）
- 用途：展示课程完整内容、媒体与互动记录
- 使用接口：
  - GET /api/v1/courses/:slug        → 获取单个课程元信息
  - GET /api/v1/content/:slug        → 获取课程标准化内容 JSON
  - POST /api/v1/statement           → 提交学习行为（xAPI）
  - GET /api/v1/media/:filename      → 获取媒体资源直链（可选）
